В XML и JSON 3 поля
SourceDirectory - исходная директория 
TargetDirectory - целевая дирректория(где хранятся архивы)
ArchiveDirectory - директория где хранятся по структуре год/месяц/день(дата закидывания файла определяется в программе и потом добавляется к строке)
